🚀 Move Compiler WASM

Browser-Native Move Compilation Demos

✅ Filesystem-Free | Zero Dependencies
What is this?

A complete Move language compiler running entirely in your browser using WebAssembly. No server required, no temporary files, pure in-memory compilation.

  • Compiles Move code to bytecode in milliseconds
  • Works completely offline
  • Full Move compiler features (type checking, optimization)
  • Production-ready and tested
🧪
E2E Test Suite
Comprehensive end-to-end tests covering basic compilation, error handling, and advanced Move features.
⚡
Benchmarks
Performance benchmarks measuring compilation speed, throughput, and resource usage across different code sizes.
💻
Interactive IDE
Write and compile Move code in an interactive editor with real-time error feedback and bytecode output.
How to Use

First, build the WASM package: 

cd aptos-move/move-compiler-wasm
wasm-pack build --target web --release

Then serve this directory and open in your browser: 

python3 -m http.server 8000
# Open http://localhost:8000/demo/demos.html
Technical Details
  • WASM Binary Size: ~2-3MB (gzipped: ~600KB)
  • Compilation Speed: 50-200ms for typical modules
  • Throughput: 2000-5000 LOC/second
  • Browser Support: Chrome, Firefox, Safari, Edge (latest)
